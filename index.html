Lesson 0: Basics of the the World Wide Web

The Web

The Web is a collection of Documents written in HTML. HTML stands for HyperText Markup Language, and is the basis for each document on the web. The web works by your browser requesting a document on a server over HTTP. This is done either via the internet or a local network.

HTML

HTML is made up of text content, markup(how its arrianged), audio/video and links. Markup is made with the follow syntax

<name of tag>Contents</name of tag>

In some cases tags tags may be void as they have no content. An example of this would be the code used to insert a picture into an html document

Attributes

Attributes are items that can be attached to tags. The syntax is as follows:

<name of tag attribute ="value">Contents</name of tag>

Inline vs Block

Inline and Block is a way to format text on a document. A block element takes up the width and height available. Block elements provides line breaks before and after the content. This can be further customized by specifing height and width. Inline is simply a line break.

Span vs DIV

Span is a inline element. Div is a block element. Both have a similar syntax:

<span class="foo">Content</span>

<div class="bar">Content</div>
